# 5-5 $And, $Or, Implicit Vs Explicit (Logical Query Operator)

- Implicit AND

```
db.firstTest.find(
    {
        age: { $gte: 18, $lt: 30 }
    },
    { age: 1 }
).sort({age: 1})
```

- Implicit AND

```
db.firstTest.find(
    {
        age: { $gte: 15, $lte: 30 }
    },
    { age: 1 }
).sort({age: 1})
```

- Explicit AND

```
db.firstTest.find(
    {
        $and: [
            { age: { $gte: 15 } },
            { age: { $lte: 30 } }
        ]
    },
    { age: 1 }
).sort({ age: 1 })
```

```
db.firstTest.find(
    {
        $and:
            [
                { age: { $ne: 15 } },
                { age: { $lt: 30 } }
            ]
    }
).project({ age: 1 })
```

```
db.firstTest.find(
    {
        $and:
            [
                { age: { $ne: 15 } },
                { age: { $lt: 30 } }
            ]
    }
).project({ age: 1 }).sort({ age: 1 })
```

```
db.firstTest.find(
    {
        $and:
            [
                { gender: 'Female' },
                { age: { $ne: 15 } },
                { age: { $lt: 30 } }
            ]
    }
).project({ age: 1, gender: 1 }).sort({ age: 1 })
```

- Explicit OR and Explicit AND:

```
db.firstTest.find(
    {
        $or: [
            { interests: "Gaming" },
            { interests: "Cooking" }
        ]
    }
).project({interests: 1})
```

```
db.firstTest.find(
    {
        $and: [
            { interests: "Gaming" },
            { interests: "Cooking" }
        ]
    }
).project({interests: 1})
```

- Implicit OR

```
db.firstTest.find(
    {
        interests: { $in: ['Cooking', 'Gaming'] }
    },
    { interests: 1 }
).sort({ age: 1 })
```

- Explicit OR

```
db.firstTest.find(
    {
        $or:
        [
            { interests: "Traveling" },
            { interests: "Cooking" }
        ]
    }
).project({ interests: 1 })
```

- implicit OR

```
db.firstTest.find(
    {
        'skills.name': {
            $in: ["JAVASCRIPT", "PYTHON"]
        }
    }
).project({ skills: 1 })
```

- Explicit OR

```
db.firstTest.find(
    {
        $or:
            [
                { 'skills.name': "JAVASCRIPT" },
                { 'skills.name': "PYTHON" },
            ]
    }
).project({ skills: 1 })
```
